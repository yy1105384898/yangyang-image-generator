Sub2API Connections

Sub2API 同步

参考 chatgpt2api 流程：保存连接，读取 OpenAI OAuth 账号，勾选账号后后台拉取 access_token 导入本地号池。

连接信息

同步说明

读取列表时只展示账号摘要；点击导入后，后端才会逐个读取账号详情里的 access_token，并写入本地号池。

支持两种认证：管理员邮箱密码登录，或直接使用 Sub2API Admin API Key。

如果 YANGYANGAPI 和 Sub2API 都在同一台 Docker 主机，服务地址可优先填 http://host.docker.internal:8091；填宿主机局域网 IP 时请确认 NAS 防火墙和 Docker 网络允许容器访问。

导入后可回到“号池管理”刷新邮箱、套餐和生图额度。
